# Set


### Definition

A flat structure containing all unique elements.


### Methods

Method      | Run time
------      | -------
add         | O(n)
remove      | O(n)
contains    | O(n)
union       | O(nm)
intersect   | O(nm)
difference  | O(nm)
isSubsetOf  | O(m)
length      | O(n)
toString    | O(1)
