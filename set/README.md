# Set


### Definition

A flat structure containing all unique elements.


### Methods

Method      | Run time
------      | -------
add         | O(n)
remove      | O(n)
contains    | O(n)
union       | O(nm)
intersect   | O(nm)
difference  | O(nm)
isSubsetOf  | O(n)
length      | O(n)
toString    | O(n)
toArray     | O(1)
